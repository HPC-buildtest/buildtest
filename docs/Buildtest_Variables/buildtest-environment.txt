.. table: buildtest Environment Variables

+--------------------------------+-----------------------------------------------------------+------------------------------+
|  Variables                     |               Description                                 |   Values                     |
+================================+===========================================================+==============================+
| BUILDTEST_CONFIGS_REPO         |    root of buildtest-configs repo                         |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_PYTHON_REPO          |    root of Python-buildtest-config repo                   |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_PERL_REPO	         |    root of Perl-buildtest-config repo                     |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_R_REPO               |    root of R-buildtest-config repo                        |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_RUBY_REPO            |    root of Ruby-buildtest-config repo                     |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_CLEAN_BUILD          |    deletes test directory before writing test scripts     |  ``True``, ``False``         |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_LOGDIR               |    directory where buildtest logs will be written         |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_TESTDIR              |    directory where test scripts will be generated         |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_MODULE_ROOT          |    colon separated list of root of module trees           |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_MODULE_NAMING_SCHEME |   Specify module naming scheme used                       |  ``FNS``, ``HMNS``           |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_SHELL                |    shell type to use when generating tests                | ``shell``, ``csh``, ``bash`` |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_ENABLE_JOB           |    enable job script integration with buildtest           |  ``True``, ``False``         |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_JOB_TEMPLATE         |    path to job template                                   |  ``<filepath>``              |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_EASYBUILD            |    check if software is built by easybuild                |  ``True``, ``False``         |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_OHPC                 |    enable OpenHPC integration with buildtest              |  ``True``, ``False``         |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_RUN_DIR              |    directory where buildtest will write `.run` files      |  ``<directory>``             |
+--------------------------------+-----------------------------------------------------------+------------------------------+
| BUILDTEST_SUCCESS_THRESHOLD    |    criteria for success threshold when running test       |  ``[0.0-1.0]``               |
+--------------------------------+-----------------------------------------------------------+------------------------------+
