usage: buildtest [options] [COMMANDS]

optional arguments:
  -h, --help            show this help message and exit

    show        Options for displaying buildtest configuration  
    testconfigs Options for list, view, and edit test configuration         
    build       Options for building test scripts  
    benchmark   Run Benchmark
    module      Module Operations including module load testing, module collections, module tree, easybuild/spack modules, parent-modules. 
    system      System Configuration
    config      Buildtest Configuration

  {list,build,show,benchmark,testconfigs,module,system,config}
