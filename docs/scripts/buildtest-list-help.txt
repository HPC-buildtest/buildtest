buildtest list --help
usage: buildtest list [-h] [-s] [-m] [-ec] [-v {all,current}]

optional arguments:
  -h, --help            show this help message and exit
  -s, --software        get unique software from Lmod spider command
  -m, --modules         get full module name and path to module files
  -ec, --easyconfigs    Return a list of easyconfigs from a module tree
  -v {all,current}, --view {all,current}
                        Control output of spider command. all: Retrieve all records current: Retrieve record where
                        module path is in BUILDTEST_MODULEPATH
