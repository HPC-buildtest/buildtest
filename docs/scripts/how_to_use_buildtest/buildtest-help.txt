usage: buildtest [-h] [-V] [--logdir LOGDIR] [--clean-logs] {list,find,yaml,build,run,module,show,benchmark} ...

buildtest is a software testing framework designed for HPC facilities to verify their Software Stack. buildtest abstracts test complexity into YAML files that is interpretedby buildtest into shell script

optional arguments:
  -h, --help            show this help message and exit

commands:

  Info:
      list        Options for listing software, module files, and  easyconfigs
      show        Options for displaying buildtest configuration
      find        Find configuration files and test scripts


  Build:
      build       Options for building test scripts
      run         Run Tests
      benchmark   Run Benchmark

  Misc:
      yaml        Yaml commands for buildtest
      module      Options for module load testing and report diff between module trees


  {list,find,yaml,build,run,module,show,benchmark}

Miscellaneous Options :
  -V, --version         show program version number and exit
  --logdir LOGDIR       Path to write buildtest logs. Override configuration BUILDTEST_LOGDIR
  --clean-logs          delete buildtest log directory $BUILDTEST_LOGDIR

Documentation: https://buildtest.readthedocs.io/en/latest/index.html
